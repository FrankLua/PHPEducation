Основы html

  1. HTML Состоит из Body и head
  2. В тег "html" является самым главным в документе
  3. В тег "head" содержит информацию о документе
  4. В тег "body" содержит страницу

Head

  1. тег meta содержит информацию о кодировке в атриббуте charset (свойство тега) обычно это utf - 8
  2. Тег link указывает иконку к сайту. Атрибут href

Теги

  1. теги заголовка h1-h6 - это блочный элемент| p тег абзаца | тег b позволяет сделать текст внутри тега жирным
  2. Тег a это сслыка| Можно использовать как якорь на свои теги. для этого нужно дать тегу id и в href оставить ссылку на него через #
  3. маркированный список ul
  4. hr блочны тег создаёт черту снизу
  5. Структура таблицы table => thead шапка => tbody => tfoot tr тег- строчки таблицы, th (жирный) или td(не жирный) это столбцы
